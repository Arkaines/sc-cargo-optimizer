Politique de confidentialité

Dernière mise à jour : 15 juillet 2026

Cette page explique quelles données l'Optimiseur de routes cargo (le "site", l'"outil") collecte, pourquoi, et comment elles sont utilisées. C'est un projet de fan gratuit et open source, sans finalité commerciale — cette politique reflète honnêtement ce que fait le code, pas un texte juridique générique.

Responsable du traitement

L'outil est développé et maintenu par Arkaines (auteur du projet), seul responsable des données décrites ci-dessous. Contact : voir la section "Contact" en bas de page.

1. Ce qui est stocké localement (toujours)

Sans te connecter avec Discord, l'outil fonctionne entièrement dans ton navigateur : tes missions, lieux personnalisés, distances manuelles, vaisseau sélectionné et calibrages de réputation sont enregistrés uniquement dans le stockage local (localStorage) de ton navigateur, sur ton propre appareil. Rien n'est envoyé à un serveur. Tu peux tout effacer à tout moment via le bouton "Réinitialiser toutes les données" de l'outil.

2. Ce qui est collecté si tu te connectes avec Discord

La connexion Discord est entièrement optionnelle et sert uniquement à synchroniser tes données de jeu (missions, lieux, distances, réputation, vaisseau) entre plusieurs appareils. Elle passe par Discord (fournisseur d'identification) puis Supabase (hébergement de la base de données). En te connectant, les données suivantes sont traitées :

  • Ton identifiant Discord, ton nom d'utilisateur et ton avatar (fournis par Discord via l'authentification), pour t'afficher connecté et associer tes données cloud à ton compte.
  • Les mêmes données de jeu que ci-dessus (missions, lieux personnalisés, distances, vaisseau, calibrages de réputation), copiées vers une base de données Supabase pour permettre la synchronisation entre appareils.

Aucune autre information Discord n'est demandée ou stockée (pas d'accès à tes serveurs, salons, messages ou contacts). Ces données ne sont jamais vendues, partagées avec des tiers à des fins publicitaires, ni utilisées pour du profilage.

Base légale (RGPD) : le traitement des données locales repose sur l'exécution du service demandé (faire fonctionner l'outil). Le traitement des données liées à la connexion Discord (identité + synchronisation cloud) repose sur ton consentement, donné en cliquant sur "Se connecter avec Discord" — tu peux le retirer à tout moment en te déconnectant et en demandant la suppression de tes données cloud (voir "Tes droits" ci-dessous).

3. Services tiers utilisés

  • Discord — authentification uniquement (OAuth), pour la connexion optionnelle. Société basée aux États-Unis. Voir la politique de confidentialité de Discord.
  • Supabase — hébergement de la base de données pour la sauvegarde cloud optionnelle. Le projet utilisé par cet outil est hébergé dans l'Union européenne. Voir la politique de confidentialité de Supabase.
  • UEX Corp API et Star Citizen Wiki API — données publiques du jeu (lieux, marchandises, missions), consultées en lecture seule ; aucune donnée personnelle ne leur est transmise.
  • Tesseract.js (reconnaissance de texte) — s'exécute entièrement dans ton navigateur. Les captures d'écran que tu importes ne sont jamais envoyées à un serveur, y compris le nôtre.
  • Cloudflare Web Analytics — mesure d'audience anonyme (pages vues, provenance), sans cookies ni identifiant qui te suivrait d'un site à l'autre. Réseau mondial (États-Unis et autres pays). Voir la politique de confidentialité de Cloudflare.

Aucune publicité et aucun outil de suivi publicitaire ou de profilage n'est utilisé sur ce site.

3bis. Transferts de données hors Union européenne

Les données de jeu synchronisées (missions, lieux, etc.) restent hébergées dans l'UE (Supabase). En revanche, Discord (authentification) et Cloudflare (mesure d'audience) sont basés aux États-Unis et peuvent traiter certaines données (ex : ton identifiant Discord, l'adresse IP utilisée pour charger la page) en dehors de l'UE, dans le cadre de leurs propres garanties de conformité (clauses contractuelles types ou équivalent). Voir leurs politiques respectives liées ci-dessus pour le détail.

4. Conservation et suppression des données

Les données locales restent sur ton appareil jusqu'à ce que tu les effaces (bouton de réinitialisation, ou effacement du stockage local du navigateur). Les données cloud (si tu t'es connecté) sont conservées tant que ton compte existe ; tu peux en demander la suppression complète à tout moment (voir "Contact" ci-dessous).

5. Tes droits (RGPD)

Si tu résides dans l'Union européenne (ou un pays avec une législation équivalente), tu disposes des droits suivants sur tes données :

  • Accès : savoir quelles données te concernant sont stockées.
  • Rectification : corriger une donnée inexacte.
  • Effacement : demander la suppression de tes données cloud (les données locales, tu les effaces toi-même à tout moment).
  • Limitation et opposition au traitement, dans les cas prévus par le RGPD.
  • Portabilité : recevoir tes données cloud dans un format exploitable.
  • Réclamation : déposer une plainte auprès de la CNIL (autorité française de protection des données) ou de l'autorité équivalente de ton pays de résidence.

Pour exercer ces droits, voir "Contact" ci-dessous.

6. Enfants

Cet outil n'est pas destiné aux personnes n'ayant pas l'âge minimum requis par les conditions d'utilisation de Discord pour créer un compte, ni, pour la connexion Discord, aux personnes n'ayant pas atteint l'âge de consentement numérique applicable dans leur pays de résidence.

7. Transparence

Le code source de cet outil est entièrement public et open source (licence MIT) : github.com/Arkaines/sc-cargo-optimizer. Tu peux y vérifier exactement quelles données sont lues, stockées ou envoyées.

8. Modifications

Cette page peut être mise à jour si les fonctionnalités de l'outil évoluent. La date de dernière mise à jour est indiquée en haut de la page.

9. Contact

Pour exercer tes droits ou pour toute question sur cette politique, ouvre une issue sur GitHub. Comme ce dépôt est public, n'y mets pas d'information sensible : indique simplement les derniers chiffres de ton identifiant Discord (visible dans tes paramètres Discord) pour que ta demande puisse être identifiée, sans exposer publiquement plus que nécessaire.
